사용 방법
  1. 상속 트리에서 클래스를 선택합니다.
  2. `PersonADT`는 수정할 수 없고, 나머지 클래스는 Python 코드를 수정할 수 있습니다.
  3. `main 코드`에서 생성자 매개변수를 바꿉니다.
  4. 오른쪽 버튼을 눌러 오버라이딩 결과와 다형성 결과를 확인합니다.

다형성 비교

PersonADT는 함수 이름만 정의하는 ADT 역할입니다. Student, Professor, Friend는 같은 함수명을 서로 다른 결과로 오버라이딩합니다.
  • 상속 관계 시각화
  • 오버라이딩 함수 버튼 실행
  • 다형성 결과 출력
